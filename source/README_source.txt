This is a description of the various Fortran source codes in /source/

BRStateFcn_params is a library for running the Bean-Rodbell model including a finite pressure

DemagField is a library for calculating the magnetic field of a magnetostatic assembly of known tiles

deprecated includes old (unused and non-concurrent) code

MagneticForceIntegrator is the library for calculating forces on a magnet

MagTense_StandAlone is a library for making a stand-alone version of the MagTense framework (an .exe file)

MagTenseFortranCuda is the link to CUDA for the Micromagnetism part of the framework

MagTenseMEX is a pseudo-project for maintaining the entry-mex-fortran files that are compiled with buildMagTenseMEX.m in the root

MagTenseMicroMag is the micromagnetic part of the MagTense framework

NumericalIntegrationlib is a dedicated library for doing numerical integrals (in 1D and 2D)

NumericalRecipes includes (some of ) the NR library

TileDemagTensor contains all the analytical / semi-analytical implementations of the demagnetization tensor field for the various tile geometries
